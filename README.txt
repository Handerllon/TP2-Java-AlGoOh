# TP2-Java-AlGoOh

TP final del curso 75.07 Algoritmos y Programación III de la Facultad de Ingeniería de la Universidad de Buenos Aires.

Consiste en implementar el juego de carta Yu-Gi-Oh utilizando Java y JavaFX.

# Cómo ejecutar el juego?

1. Abrir una terminal en el directorio raiz del repositorio.

2. Ejecutar el comando:
$ ant

3. Una vez que descargó las dependencias, ejecutar el comando
$ ant ejecutar.aplicacion

4. Jugar el juego.
